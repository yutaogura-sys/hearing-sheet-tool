ヒアリングシート → スプレッドシート 変換ツール

使い方ガイド

ツールを開く

このツールでできること

PDFのヒアリングシートをAIが自動で読み取って、スプレッドシートにそのまま貼り付けられるデータに変換します。

手書きの文字もAIが読み取ります。18項目中16項目以上を自動入力できます。

はじめに：AIキーの取得（初回のみ・5分で完了）
このツールを使うには「Gemini APIキー」が必要です。
Googleが無料で提供しているAI機能の利用キーです。
最初に1回だけ取得すれば、あとはずっと使えます。

1下のリンクをクリック

まず、GoogleのAIサービスのページを開きます。

Google AI Studio を開く

Googleのログイン画面が出たら、いつも使っているGoogleアカウント（Gmailのアカウント）でログインしてください。
▼

2「APIキーを作成」ボタンをクリック

ページが開いたら、画面の中にある「APIキーを作成」（または「Create API Key」）というボタンを探してクリックします。

  1. 「APIキーを作成」ボタンをクリック
  2. 「新しいプロジェクトで APIキーを作成」を選ぶ（どれを選んでもOK）
  3. 少し待つと、長い英数字の文字列が表示されます
表示された文字列が「APIキー」です。
AIzaSy... のような文字列です。
「コピー」ボタンを押して、この文字列をコピーしてください。
お金はかかりません。Gemini APIは無料で使えます。クレジットカードの登録も不要です。
▼

3ツールにAPIキーを貼り付ける

コピーしたAPIキーを、変換ツールに貼り付けます。

  1. フォルダ内の 「★ここをダブルクリックで起動★」 をダブルクリックしてツールを開く
  2. 画面の上のほうにある 「接続設定」 をクリックして開く
  3. 「Gemini APIキー」 と書かれた入力欄に、コピーしたキーを貼り付ける（Ctrl+V）
  4. 「接続テスト」 ボタンを押す
「Gemini API 接続OK」と表示されれば成功です！
これで準備完了です。次回からはキーの入力は不要です（ブラウザが記憶します）。
うまくいかない場合: APIキーの前後に余分なスペースが入っていないか確認してください。もう一度コピーし直して貼り付けてみてください。
ツールの使い方（毎回の操作）

1PDFを読み込ませる

  1. フォルダ内の 「★ここをダブルクリックで起動★」 をダブルクリックしてツールを開く
  2. 画面の真ん中にある点線の枠に、PDFファイルをドラッグ&ドロップする
    （または枠をクリックしてファイルを選択）
  3. 右下の 「次へ：OCR解析 →」 ボタンをクリック
  4. AIが読み取りを開始します（10〜30秒ほど待ちます）
▼

2読み取り結果を確認する

AIが読み取った内容が画面に表示されます。

  • 黄色の背景 のところ = AIが自動で入力した項目です
  • O（緑の丸）= 読み取り成功した項目
  • X（赤のバツ）= 読み取れなかった項目 → 手入力が必要
  • AIが読み取った内容が正しいか、必ず目で確認してください
  • 間違っている場合は、その欄をクリックして修正できます
▼

3スプレッドシートに貼り付ける

  1. 確認が終わったら 「コピー画面へ →」 をクリック
  2. 緑色の大きなボタン 「A〜V列を一行コピー」 をクリック
    （データがクリップボードにコピーされます）
  3. 案件管理表のスプレッドシートを開く
  4. データを入れたい行の A列 をクリック
  5. Ctrl+V（貼り付け）を押す
これで完了です！ スプレッドシートにデータが入力されます。

よくある質問

Q. APIキーって何ですか？安全ですか？

APIキーは、GoogleのAIサービスを使うための「合鍵」のようなものです。あなたのブラウザの中だけに保存され、Google以外には送信されません。安心して使えます。

Q. 本当に無料ですか？

はい、無料です。Gemini APIには無料枠があり、1日に数十件のヒアリングシートを読み取っても料金はかかりません。クレジットカードの登録も不要です。

Q. 「接続テスト」でエラーが出ます

以下を確認してください:

  • APIキーの前後に余分なスペースが入っていませんか？
  • APIキーを途中までしかコピーしていませんか？ → もう一度コピーし直してください
  • インターネットに接続されていますか？

Q. 次回もAPIキーを入れる必要がありますか？

不要です。一度入力すれば、ブラウザが記憶するので次回以降は自動で読み込まれます。ただし、ブラウザのデータを消去した場合は再入力が必要です。

Q. 読み取り精度はどのくらいですか？

Gemini AI使用時は、18項目中16項目以上を自動認識できます。ただし手書き文字の個人差や、記入内容によっては読み取れない項目もあります。必ず確認・修正してからスプレッドシートに貼り付けてください。

Q. 複数のPDFをまとめて処理できますか？

はい、できます。ファイル選択時に複数のPDFをドラッグ&ドロップすれば、タブで切り替えながら確認できます。

Q. チームで同じAPIキーを共有できますか？

可能です。ただし無料枠のリクエスト数は共有になるため、利用者が多い場合は各自で取得することをおすすめします。

上級者向け：GAS設定（フォールバック用・通常不要）

Gemini APIが使えない場合のフォールバックとして、Google Drive OCRを利用するためのGAS設定です。通常は不要です。

GAS設置手順

  1. 案件管理表のスプレッドシートを開く → 拡張機能 → Apps Script
  2. 配布フォルダ内の Code.gs の内容を貼り付けて保存
  3. 左メニュー「サービス」→「+」→「Drive API」を追加（v2）
  4. デプロイ → 新しいデプロイ → ウェブアプリ（実行ユーザー: 自分、アクセス: 全員）
  5. 表示されたURLをツールの「詳細設定」→「GAS URL」欄に貼り付け
